# Lutris Kodi Addon

Launch games through [Lutris](http://lutris.net).

## Install

1. Download the addon to ~/.kodi/addons/script.lutris/
  ```
  git clone https://github.com/RobLoach/script.lutris.git ~/.kodi/addons/script.lutris
  ```
2. Enable the addon in Kodi
